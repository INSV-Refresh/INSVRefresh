Política de Privacidade

Data de Vigência: 15 de março de 2026 Última Atualização: 15 de março de 2026

Em resumo: O INSV Refresh não coleta, armazena, vende ou compartilha nenhum dado pessoal dos usuários. Todas as configurações são armazenadas localmente no seu navegador e nunca saem do seu dispositivo.

1. Introdução

Esta Política de Privacidade descreve como a extensão INSV Refresh ("nós", "nossa" ou "extensão") trata informações relacionadas ao uso da extensão do navegador Chrome.

O INSV Refresh é uma ferramenta de produtividade projetada para monitorar filas do Salesforce Lightning e notificar usuários sobre novos chamados, sem coletar ou transmitir qualquer informação pessoal.

1.1. Definição de Dados Pessoais

Para os fins desta política, "dados pessoais" são quaisquer informações relacionadas a uma pessoa natural identificada ou identificável, conforme definido pela Lei Geral de Proteção de Dados (LGPD) do Brasil e pelo Regulamento Geral sobre a Proteção de Dados (GDPR) da União Europeia. Isso inclui, mas não se limita a: nome, endereço de e-mail, endereço IP, identificadores de dispositivo, dados de localização, cookies identificadores e quaisquer outras informações que possam ser usadas para identificar direta ou indiretamente um indivíduo.

2. Informações que NÃO Coletamos

A extensão INSV Refresh não coleta, armazena, processa ou transmite nenhum dos seguintes tipos de dados pessoais:

  • Informações de identificação pessoal (nome, e-mail, telefone, endereço)
  • Dados de autenticação (senhas, tokens, credenciais)
  • Histórico de navegação
  • Conteúdo de páginas web visitadas
  • Dados do Salesforce (registros, casos, informações de clientes)
  • Localização geográfica
  • Informações financeiras ou de pagamento
  • Endereços IP ou identificadores de dispositivo
  • Cookies de rastreamento ou identificação

3. Dados Armazenados Localmente

A extensão armazena as seguintes configurações exclusivamente no seu navegador usando a API chrome.storage. Estes dados nunca saem do seu dispositivo:

3.1. Configurações de Monitoramento

  • Nomes de filas: Os nomes das filas do Salesforce que você escolheu monitorar
  • Intervalos de atualização: A frequência (em segundos) com que cada fila deve ser atualizada
  • Status de ativação: Quais filas estão ativas ou inativas

3.2. Preferências de Notificação

  • Sons de notificação: Qual som usar para alertas (padrão ou personalizado)
  • Volume: Nível de volume das notificações sonoras
  • Áudios personalizados: Arquivos de áudio que você escolheu fazer upload (armazenados como base64 no navegador)
  • Configurações de notificação de status: Alertas para mudanças de status de chamados

3.3. Configurações de Interface

  • Modo escuro: Preferência de tema claro/escuro
  • Modo Legacy: Se o modo simplificado está ativado
  • Atalhos de teclado: Teclas configuradas para ações rápidas

3.4. Retenção de Dados Locais

Os dados armazenados localmente permanecem no seu navegador enquanto a extensão estiver instalada. Quando você desinstala a extensão, todos os dados são automaticamente removidos pelo Chrome. Se você reinstalar a extensão, será necessário reconfigurar suas preferências, pois os dados anteriores terão sido apagados.

Importante sobre chrome.storage.sync: Quando você usa chrome.storage.sync (para sincronizar configurações entre seus dispositivos), os dados são sincronizados através dos servidores do Google. Neste caso:

  • A extensão INSV Refresh não envia esses dados diretamente para nossos servidores
  • A sincronização é gerenciada inteiramente pelo Google Chrome
  • A Política de Privacidade do Google se aplica a esses dados sincronizados
  • Você já consentiu com essa sincronização ao usar o Chrome e fazer login com sua conta Google

4. Consentimento

Ao instalar e utilizar a extensão INSV Refresh, você consente com:

  • O armazenamento local das suas configurações no navegador
  • A sincronização dessas configurações via chrome.storage.sync (gerenciada pelo Google)
  • A injeção de scripts nas páginas do Salesforce Lightning para permitir o funcionamento da extensão

Você pode revogar esse consentimento a qualquer momento desinstalando a extensão. Todos os dados armazenados localmente serão automaticamente removidos.

5. Permissões da Extensão

A extensão solicita as seguintes permissões do Chrome e como elas são usadas:

5.1. storage

Usada para salvar suas configurações localmente no navegador. Nenhum dado sai do seu dispositivo, exceto via chrome.storage.sync que sincroniza entre seus próprios dispositivos Google (conforme explicado na seção 3.4).

5.2. scripting

Usada para injetar código nas páginas do Salesforce Lightning (*.lightning.force.com) para:

  • Detectar o nome da fila atual
  • Monitorar o contador de chamados
  • Executar atualizações da lista usando a API nativa do Salesforce ($A.get('e.force:refreshView').fire())

Este código funciona apenas em páginas do Salesforce e não afeta outros sites.

5.3. Host permission (*.lightning.force.com)

Permite que a extensão funcione apenas em páginas do Salesforce Lightning. A extensão permanece completamente inativa em todos os outros sites.

6. Código de Terceiros

A extensão utiliza os seguintes recursos externos:

6.1. Google Fonts

Utilizamos fontes tipográficas do Google Fonts (fonts.googleapis.com) para melhorar a aparência visual da interface de configurações. Apenas arquivos CSS e de fonte (.woff2) são carregados, sem execução de JavaScript externo.

A Política de Privacidade do Google se aplica ao uso do Google Fonts.

6.2. ExtensionPay e Stripe

Para usuários que optam pela assinatura premium, utilizamos:

  • ExtensionPay: Plataforma de gerenciamento de assinaturas para extensões. O ExtensionPay não coleta dados de navegação e mantém os usuários anônimos até o momento do pagamento, armazenando apenas o e-mail do usuário e o status da assinatura.
  • Stripe: Processador de pagamentos que lida com todas as transações financeiras e armazenamento seguro de informações de cartão de crédito.

Importante:

  • Os dados de pagamento são processados diretamente pelo Stripe, não pela extensão ou pelo ExtensionPay
  • A extensão recebe apenas um indicador de status de assinatura (ativo/inativo)
  • Nenhum dado financeiro ou pessoal é acessado ou armazenado pela extensão
  • O ExtensionPay armazena apenas seu e-mail e status de assinatura, sem coletar dados de navegação

Políticas de privacidade e informações relevantes:

  • As informações sobre o tratamento de dados pelo ExtensionPay podem ser consultadas na seção de FAQ (User Privacy) em sua página oficial
  • Stripe Privacy Policy

7. Transferência Internacional de Dados

Como a extensão não coleta dados pessoais diretamente, não há transferência internacional de dados pela extensão. No entanto:

  • Ao usar chrome.storage.sync, os dados podem ser sincronizados através dos servidores do Google, que podem estar localizados em diferentes jurisdições. Esta sincronização é regida pela Política de Privacidade do Google.
  • Ao usar Google Fonts, solicitações são feitas aos servidores do Google para carregar os arquivos de fonte.
  • Ao processar pagamentos via ExtensionPay/Stripe, os dados são tratados de acordo com as políticas desses serviços, sendo que o Stripe é o responsável final pelo processamento financeiro e armazenamento seguro de dados sensíveis de pagamento.

8. Compartilhamento de Dados

Nós não compartilhamos, vendemos ou transferimos seus dados a terceiros, pois não coletamos dados pessoais em primeiro lugar.

Os únicos dados que transitam entre sistemas são:

  • Configurações sincronizadas via chrome.storage.sync (gerenciadas pelo Google)
  • Status de assinatura via ExtensionPay (apenas indicador ativo/inativo)

9. Segurança

Como não coletamos dados pessoais, não há risco de vazamento de informações sensíveis dos nossos sistemas. Todas as configurações são armazenadas localmente no seu navegador usando APIs seguras do Chrome.

Recomendamos que você:

  • Mantenha seu navegador Chrome atualizado
  • Use senhas fortes para sua conta Google (se usar chrome.storage.sync)
  • Revise regularmente as extensões instaladas

10. Seus Direitos

Como todos os dados são armazenados localmente no seu navegador, você tem controle total:

  • Acesso: Todos os dados estão disponíveis nas configurações da extensão
  • Modificação: Você pode alterar qualquer configuração a qualquer momento
  • Exclusão: Desinstalar a extensão remove todos os dados armazenados automaticamente
  • Exportação: Use a função "Exportar Configurações" para fazer backup dos seus dados em formato JSON
  • Portabilidade: Os dados exportados podem ser importados novamente através da função "Importar Configurações"

11. Privacidade de Crianças

A extensão não é direcionada a menores de 13 anos e não coleta intencionalmente informações de crianças. Se você é pai ou responsável e acredita que uma criança forneceu informações para a extensão (embora isso não seja possível dado que não coletamos dados), entre em contato conosco.

12. Alterações nesta Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Mudanças significativas serão comunicadas através de:

  • Atualização da data "Última atualização" no topo desta página
  • Notificação na página de configurações da extensão
  • Changelog da versão na Chrome Web Store

Recomendamos que você revise esta política periodicamente para se manter informado sobre como protegemos sua privacidade.

13. Conformidade Legal

  • Lei Geral de Proteção de Dados (LGPD - Brasil, Lei nº 13.709/2018)
